# Livret d’accueil des mathématicien·nes


## Description
Livret d’accueil destiné aux participant·es de la journée d’accueil des nouvelles et nouveaux recruté·es en mathématiques. 
Cette version hébergée sur GitHub est une version publique. Une autre version existe sur le GitLab de la PLM.

## Dernière version à jour
À chaque modification des sources sur le GitLab de la PLM, le livret est compilé, et le pdf peut être obtenu à l’adresse suivante :
https://plmlab.math.cnrs.fr/frouvelle/livret-d-accueil-des-mathematicien-nes/-/jobs/artifacts/main/raw/livret.pdf?job=build

## Contribution
N’hésitez pas à signaler des liens rompus ou des informations nécessitant une mise à jour. Soit en faisant une « Issue », ou simplement en proposant un « Merge Request ».
